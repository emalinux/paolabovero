# ✨ Paola Bovero — Luce · Scrittura · Artigianato

Sito ufficiale di **Paola Bovero**.  
Un percorso tra parola, luce e creazione: libri, candele e sentieri interiori raccontati in uno spazio digitale progettato su misura.

[![Netlify Status](https://api.netlify.com/api/v1/badges/1cd1aa59-48bd-4655-8207-896e70fad7a6/deploy-status)](https://app.netlify.com/projects/demoraggidiluce/deploys)

---

## 🔧 Stack Tecnico

- **Generator:** [Hugo](https://gohugo.io/) `v0.121.x`
- **Hosting/CDN:** [Netlify](https://www.netlify.com)
- **Repository:** [GitHub](https://github.com/emalinux/paolabovero)
- **Tema:** Custom sviluppato da zero
- **CSS:** Interamente scritto a mano
- **JavaScript:** Vanilla (lightbox custom)

---

## 🧱 Filosofia di sviluppo

Questo progetto non utilizza temi preconfezionati né builder visuali.

- Layout progettato su carta
- Struttura narrativa a scorrimento
- Architettura modulare
- Performance-first
- Nessun framework CSS esterno

Ogni sezione è stata costruita per sostenere l'identità dell’autrice, non per adattarsi a un template generico.

---

## ⚙️ Deploy automatico

- Trigger: ogni `git push` su `main`
- Build command: `hugo`
- Publish directory: `public`
- Variabile ambiente:
  - `HUGO_VERSION`

Dominio demo:
👉 https://demoraggidiluce.netlify.app
